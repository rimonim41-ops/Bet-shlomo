# Gifté Studio – אתר מתנות מעוצב

זהו דף נחיתה/אתר תדמית למותג מתנות בעיצוב אישי, כולל:

- Hero שיווקי עם קריאה לפעולה
- כרטיסיות מכירה למוצרים מובילים
- אזור "כל מה שאתר כזה צריך"
- המלצת לקוח
- אזור יצירת קשר עם CTA לוואטסאפ
- עיצוב רספונסיבי לנייד

## איך להעלות ל-GitHub כפרויקט חדש

1. יוצרים ריפו חדש ב-GitHub (למשל: `gifte-studio`).
2. מריצים מקומית:

```bash
git remote add origin https://github.com/<your-user>/gifte-studio.git
git branch -M main
git push -u origin main
```

## קבצים מרכזיים

- `index.html` – כל האתר (HTML + CSS)
- קבצי תמונות (`*.jpg`, `*.jpeg`) – תמונות לתצוגת מוצרים
